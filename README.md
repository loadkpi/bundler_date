# BundlerDate

## Installation

Specify the plugin in your Gemfile:

```ruby
plugin 'bundler_date'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ bundle plugin install bundler_date

## Usage

    $ bundle date
